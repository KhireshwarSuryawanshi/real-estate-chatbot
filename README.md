# real-estate-chatbot
Real Estate Market Analyzer
Django + React + Pandas
A full-stack application that analyzes real-estate market data, visualizes price trends, and provides insights based on locality or city.
This project was built as part of a technical assignment.
- Tech Stack
Backend
Python
Django REST Framework
Pandas
OpenPyXL
CORS Headers
Frontend
React (Vite)
Chart.js
Fetch API
CSS

- Features
Search real estate data by locality or city
Automatic year-wise price trend chart
Detailed data table for filtered records
Smart summary generation
Fast, responsive UI (Vite + React)
Clean folder structure
Error handling for invalid queries
